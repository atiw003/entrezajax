Entrez-AJAX

An AJAX endpoint for NCBI's Entrez service

Entrez-AJAX provides an interface which returns JSON data from the NCBI Entrez service which can be accessed by a web browser directly. The interface can circumvent web browsers' same origin policies by supporting JSONP. Read the initial announcement.

What can it be used for?

The possibilities are unlimited, but to whet your appetite:

  • publication lists on your group website
  • correct spelling in user-submitted data
  • find related articles

How do I use it?

If you are developing a website that needs access to NCBI Entrez via AJAX then please register for an API key and then read the API documentation.

How does it work?

If you are interested in deploying a similar service or are just curious how it works, please read the implementation details.
