Q: Methods return a 500 error, particularly efetch

A: Google App Engine enforces a 2 megabyte fetch limit from external services such as Entrez. If this limit is hit, EntrezAJAX will return a 500 error code, as well as setting the dictionary field 'error' to be TRUE within the 'entrezajax' control structure and a descriptive error within the field 'error_message', e.g.
{'entrezajax' : {'error' : True, 'errormessage' : '<some error message>'}}
If this happens to you, consider reducing the number of results returned (if using a combined method) with the max parameter, or use the Esummary method instead.
